CAP GESTION - NICOLAS MERLIER

Contenu :
- index.html : application principale
- manifest.json : paramètres PWA
- service-worker.js : cache et fonctionnement application
- icons/ : icônes de l'application

Fonctionnalités :
- Application intitulée CAP gestion
- Choix du thème à réviser
- Questions type Kahoot
- Réponses mélangées aléatoirement
- Réponses équilibrées en longueur
- Mini-jeu Playmobil / Groot avec score
- Infos-bulles sur les notions particulières
- Glossaire des définitions
- Glossaire des formules de calcul
- Contextes adaptés : boulangerie, pâtisserie, boucherie, coiffure, cuisine, service, hôtellerie, restauration

Installation GitHub Pages :
1. Dézipper le pack.
2. Envoyer index.html, manifest.json, service-worker.js et le dossier icons à la racine du dépôt.
3. Aller dans Settings > Pages.
4. Choisir Deploy from a branch > main > root.
5. Ouvrir le lien généré.

Installation téléphone :
- iPhone : ouvrir le lien dans Safari > Partager > Sur l’écran d’accueil.
- Android : ouvrir le lien dans Chrome > menu ⋮ > Installer l’application ou Ajouter à l’écran d’accueil.
